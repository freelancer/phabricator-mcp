What is phabricator-mcp?

phabricator-mcp is an MCP server that wraps Phabricator's Conduit API, enabling AI coding assistants to interact with your Phabricator instance using natural language. Instead of switching between your editor and browser, you can ask your AI assistant to "show my assigned tasks" or "add a comment to T12345" directly.

The server implements the Model Context Protocol (MCP), an open standard developed by Anthropic for connecting AI assistants to external tools and data sources. This means phabricator-mcp works with any MCP-compatible client.

Quick Start

Claude Code (CLI)

claude mcp add --scope user phabricator -- npx github:freelancer/phabricator-mcp

Codex (OpenAI CLI)

Add to ~/.codex/config.json:

{
  "mcpServers": {
    "phabricator": {
      "command": "npx",
      "args": ["github:freelancer/phabricator-mcp"]
    }
  }
}

Configuration

The server automatically reads credentials from ~/.arcrc if you've set up Arcanist. No additional configuration is needed.

Alternatively, set environment variables (these take precedence over .arcrc):

PHABRICATOR_URL=https://phabricator.example.com
PHABRICATOR_API_TOKEN=api-xxxxx

Get an API token from your Phabricator instance at Settings > Conduit API Tokens.

Frequently Asked Questions

What is phabricator-mcp?

phabricator-mcp is an open-source MCP (Model Context Protocol) server that connects AI assistants like Claude, ChatGPT, and GitHub Copilot to Phabricator. It provides 26 tools covering task management, code reviews (including raw diff fetching), repositories, wikis, and more.

Which AI assistants work with phabricator-mcp?

phabricator-mcp works with 6 major AI coding assistants: Claude Code CLI, VS Code with Claude Extension, Cursor, GitHub Copilot, OpenAI Codex CLI, and opencode. Any MCP-compatible client can use it.

How do I install phabricator-mcp?

For Claude Code CLI, run: claude mcp add --scope user phabricator -- npx github:freelancer/phabricator-mcp. For other editors, add the MCP server configuration to your settings file. The server reads credentials from ~/.arcrc or environment variables.

What Phabricator features does phabricator-mcp support?

phabricator-mcp provides 26 tools covering: Maniphest (task search, create, edit, comment), Differential (revision search, edit, diff search, raw diff content), Diffusion (repository and commit search), Projects (search, edit, workboard columns), Phriction (wiki search and edit), Pastes (search and create), and PHID utilities.

Is phabricator-mcp free to use?

Yes, phabricator-mcp is completely free and open-source under the MIT license. You can use it, modify it, and distribute it without any cost.

What is the Model Context Protocol (MCP)?

MCP is an open standard developed by Anthropic for connecting AI assistants to external tools and data sources. It allows AI models to securely access APIs, databases, and other services through a standardized protocol.
